# React + Vite

This template provides a minimal setup to get React working in Vite with HMR (Hot Module Replacement) and some ESLint rules.

---

## How to Run the Project

Follow these steps to execute the project locally on your machine.

### Prerequisites

You must have [Node.js](https://nodejs.org) installed on your computer.

### 1. Install Dependencies
Open your terminal in the root directory of the project and run the following command to install the required packages:
```bash
npm install
```

### 2. Start the Development Server
Once the installation is complete, execute the following command to launch the local server:
```bash
npm run dev
```

### 3. Open in Your Browser
The terminal will display a local address. Open your browser and navigate to:
**http://localhost:5173** (or the alternative port displayed in your terminal)

## Other Useful Commands

* **Build for production:**
  ```bash
  npm run build
  ```
* **Preview the production build locally:**
  ```bash
  npm run preview
  ```
